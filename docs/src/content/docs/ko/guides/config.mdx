## 디버그 로깅

SDK는 디버그 로깅을 위해 [`debug`](https://www.npmjs.com/package/debug) 패키지를 사용합니다. 자세한 로그를 보려면 `DEBUG` 환경 변수를 `openai-agents*`로 설정하세요.

```bash
export DEBUG=openai-agents*
```

`@openai/agents`의 `getLogger(namespace)`를 사용해 모듈 전용 네임스페이스 로거를 얻을 수 있습니다.

<Code lang="typescript" code={getLoggerExample} title="로거 가져오기" />

### 로그의 민감한 데이터

일부 로그에는 사용자 데이터가 포함될 수 있습니다. 다음 환경 변수를 설정하여 비활성화하세요.

LLM 입력과 출력을 로깅하지 않으려면:

```bash
export OPENAI_AGENTS_DONT_LOG_MODEL_DATA=1
```

도구 입력과 출력을 로깅하지 않으려면:

```bash
export OPENAI_AGENTS_DONT_LOG_TOOL_DATA=1
```
