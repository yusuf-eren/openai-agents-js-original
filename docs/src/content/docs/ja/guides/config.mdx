## デバッグログ

SDK はデバッグロギングに [`debug`](https://www.npmjs.com/package/debug) パッケージを使用します。`DEBUG` 環境変数を `openai-agents*` に設定すると詳細ログが表示されます。

```bash
export DEBUG=openai-agents*
```

`@openai/agents` の `getLogger(namespace)` を使って、自分のモジュール用の名前空間付きロガーを取得できます。

<Code lang="typescript" code={getLoggerExample} title="ロガーの取得" />

### ログ内の機微データ

一部のログにはユーザーデータが含まれる場合があります。これらの環境変数を設定して無効化します。

LLM の入出力の記録を無効化するには:

```bash
export OPENAI_AGENTS_DONT_LOG_MODEL_DATA=1
```

ツールの入出力の記録を無効化するには:

```bash
export OPENAI_AGENTS_DONT_LOG_TOOL_DATA=1
```
